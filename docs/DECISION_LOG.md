# Decision Log

## D001

Project name changed from "Yazd Pharmacies" to "Pharmacy Location Intelligence".

Reason:
The project is designed to become reusable for any city.

---

## D002

Study area is defined as the Functional Urban Area instead of administrative city boundaries.

Reason:
Pharmacy investment depends on functional accessibility, healthcare demand, rental cost and regulations rather than municipality borders.

---

## D003

Google Maps will not be used as the primary data source.

Reason:
Cost, API limitations and licensing restrictions.

Primary data sources:

- OpenStreetMap
- Overpass API
- Neshan API (optional)
